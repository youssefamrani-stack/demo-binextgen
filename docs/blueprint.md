# **App Name**: LVMH Enlight

## Core Features:

- Conversational BI: Allow Eleonora to ask business questions using natural language.
- Data Understanding Tool: Enable the AI to understand LVMH-specific data semantics (products, sales, stores, plans). Use reasoning to decide how to respond.
- Initial Dashboard: Display a welcome message and key KPI widgets upon login, providing Eleonora a daily performance brief for Louis Vuitton.
- Follow-up Question Context: Enable the AI to maintain context from previous questions, facilitating more natural conversations.
- Multi-Agent Collaboration Tool: Use a combination of specialized AI agents (Intent, Data, Insight) to understand user intent, query data, and synthesize insights. Use reasoning to decide when or if to respond.
- Proactive Suggestions: The AI will proactively offer suggestions and insights based on the data analysis, increasing user value.
- Email Draft Action: Offer the option to draft an email based on the proactive suggestions for Eleonora to streamline her workflow.

## Style Guidelines:

- Background color: Dark theme with #1A1A2E for a sophisticated and premium feel.
- Primary color: Muted gold (#C0B283) to reflect luxury and elegance.
- Accent color: Off-white (#F5F5F5) to ensure clarity and legibility of text and charts.
- Font: 'Inter' (sans-serif) for a clean and modern look throughout the application, suitable for both headings and body text.
- Note: currently only Google Fonts are supported.
- Minimalist icons to support primary actions, with a style aligned to luxury design.
- Single-page application layout for a seamless and intuitive user experience.
- Smooth transitions and subtle animations to enhance user engagement without distracting from the data.
